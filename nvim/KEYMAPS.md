# Keymaps

`<leader>` = `Space`

## Vim 基本 — モード切替

| キー | 説明 |
|------|------|
| `i` | インサートモードへ（カーソル前） |
| `a` | インサートモードへ（カーソル後） |
| `o` | 次の行にインサートモードで入る |
| `O` | 前の行にインサートモードで入る |
| `v` | ビジュアルモードへ |
| `V` | ビジュアルライン（行選択）モードへ |
| `<ESC>` / `<D-S-j>` | ノーマルモードに戻る |

## Vim 基本 — カーソル移動

| キー | 説明 |
|------|------|
| `h` `j` `k` `l` | 左・下・上・右 |
| `w` | 次の単語の先頭へ |
| `b` | 前の単語の先頭へ |
| `e` | 次の単語の末尾へ |
| `0` | 行の先頭へ |
| `$` | 行の末尾へ |
| `gg` | ファイルの先頭へ |
| `G` | ファイルの末尾へ |
| `{` `}` | 前・次の空行へ |
| `<C-d>` | 半ページ下へ |
| `<C-u>` | 半ページ上へ |
| `f<文字>` | 行内の文字へ移動 |
| `%` | 対応する括弧へ移動 |

## Vim 基本 — 編集

| キー | 説明 |
|------|------|
| `dd` | 行を削除（カット） |
| `yy` | 行をコピー |
| `p` | 貼り付け（カーソルの後） |
| `P` | 貼り付け（カーソルの前） |
| `u` | アンドゥ |
| `<C-r>` | リドゥ |
| `cw` | 単語を変更（削除してインサート） |
| `ciw` | 単語全体を変更 |
| `ci"` | `"` 内を変更 |
| `di"` | `"` 内を削除 |
| `>>` | インデントを増やす |
| `<<` | インデントを減らす |
| `gcc` | 行をコメントトグル |
| `gc` | 選択範囲をコメントトグル (visual) |

## Vim 基本 — 検索・置換

| キー | 説明 |
|------|------|
| `/検索ワード` | 前方検索 |
| `?検索ワード` | 後方検索 |
| `n` | 次の検索結果へ |
| `N` | 前の検索結果へ |
| `:%s/旧/新/g` | ファイル全体で置換 |

## Vim 基本 — 保存・終了

| キー | 説明 |
|------|------|
| `:w` | 保存 |
| `:q` | 終了 |
| `:wq` | 保存して終了 |
| `:q!` | 強制終了 |
| `:qa` | 全ウィンドウを閉じる |
| `:qa!` | 全ウィンドウを強制終了 |

## Vim 基本 — 折りたたみ

| キー | 説明 |
|------|------|
| `zc` | 折りたたむ |
| `zo` | 展開する |
| `zM` | 全て折りたたむ |
| `zR` | 全て展開する |

## バッファ (bufferline)

| キー | 説明 |
|------|------|
| `<Tab>` | 次のバッファへ |
| `<S-Tab>` | 前のバッファへ |
| `<leader>bd` | バッファを閉じる |

## ファイル検索 (Telescope)

| キー | 説明 |
|------|------|
| `<leader>ff` | ファイル名で検索 |
| `<leader>fg` | ファイル内容をgrep |
| `<leader>fb` | 開いているバッファ一覧 |
| `<leader>fc` | 現在のバッファ内を検索 |
| `<leader>fh` | ヘルプタグ検索 |

## ファイルエクスプローラー (oil.nvim)

| キー | 説明 |
|------|------|
| `-` | 現在ファイルの親ディレクトリを開く |
| `<C-n>` | フローティングウィンドウで開く |
| `<Space>` | ディレクトリに入る / ファイルを開く |
| `<C-v>` | 垂直分割で開く |
| `<C-h>` | 水平分割で開く |
| `-` (oil内) | 親ディレクトリへ戻る |
| `q` | 閉じる |
| `<leader>cd` (oil内) | 表示中のディレクトリに `:cd` する |

## ウィンドウ操作

| キー | 説明 |
|------|------|
| `<C-\>v` | 垂直分割 |
| `<C-\>h` | 水平分割 |
| `<C-\><C-\>` | 次のウィンドウへ移動 |

## ジャンプリスト・位置移動

| キー | 説明 |
|------|------|
| `<C-o>` | ジャンプ前の位置に戻る |
| `<C-i>` | 戻った位置から進む |
| `''` | 直前にいた行へ戻る |
| ` `` ` | 直前にいた正確な位置へ戻る |
| `gi` | 最後にインサートモードだった位置へ戻る |
| `m<文字>` | 現在位置にマークを設定（例: `ma`） |
| `'<文字>` | 設定したマークへジャンプ（例: `'a`） |

## LSP — ナビゲーション

| キー | 説明 |
|------|------|
| `gd` | 定義へジャンプ (Telescope) |
| `gD` | 宣言へジャンプ |
| `gr` | 参照一覧 (Telescope) |
| `gi` | 実装へジャンプ (Telescope) |
| `gt` | 型定義へジャンプ (Telescope) |
| `K` | 折りたたみプレビュー / ドキュメント表示 |

## LSP — 編集

| キー | 説明 |

## LSP — 診断 (Diagnostics)

| キー | 説明 |

| `[d` | 前の診断へ移動 |
| `]d` | 次の診断へ移動 |
| `<leader>dl` | 診断一覧 (Telescope) |
