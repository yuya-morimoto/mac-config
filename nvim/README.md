# Neovim 設定

## 依存パッケージ

```bash
brew install nvim lazygit ripgrep
```

## プラグイン一覧

| プラグイン | 用途 |
|--------|------|
| [lazy.nvim](https://github.com/folke/lazy.nvim) | プラグインマネージャー |
| [onedark.nvim](https://github.com/navarasu/onedark.nvim) | カラーテーマ（透過対応） |
| [nvim-treesitter](https://github.com/nvim-treesitter/nvim-treesitter) | シンタックスハイライト |
| [mason.nvim](https://github.com/mason-org/mason.nvim) + [mason-lspconfig](https://github.com/mason-org/mason-lspconfig.nvim) | LSP サーバーの自動インストール |
| [nvim-lspconfig](https://github.com/neovim/nvim-lspconfig) | LSP クライアント設定 |
| [blink.cmp](https://github.com/saghen/blink.cmp) | コード補完 |
| [telescope.nvim](https://github.com/nvim-telescope/telescope.nvim) | ファジーファインダー |
| [oil.nvim](https://github.com/stevearc/oil.nvim) | ファイルエクスプローラー |
| [lualine.nvim](https://github.com/nvim-lualine/lualine.nvim) | ステータスライン |
| [bufferline.nvim](https://github.com/akinsho/bufferline.nvim) | バッファをタブ表示 |
| [which-key.nvim](https://github.com/folke/which-key.nvim) | キーマップのポップアップヒント |
| [indent-blankline.nvim](https://github.com/lukas-reineke/indent-blankline.nvim) | インデントガイド |
| [nvim-ufo](https://github.com/kevinhwang91/nvim-ufo) | コード折りたたみ |
| [lazygit.nvim](https://github.com/kdheepak/lazygit.nvim) | LazyGit 連携 |
| [render-markdown.nvim](https://github.com/MeanderingProgrammer/render-markdown.nvim) | Markdown レンダリング |
| [nvim-autopairs](https://github.com/windwp/nvim-autopairs) | 括弧の自動補完 |
| [Comment.nvim](https://github.com/numToStr/Comment.nvim) | コメントトグル |
| [vim-translator](https://github.com/voldikss/vim-translator) | 選択テキストを日本語に翻訳 |
| [blamer.nvim](https://github.com/z4p5a9/blamer.nvim) | Git blame をインライン表示 |
| [fidget.nvim](https://github.com/j-hui/fidget.nvim) | LSP 進捗を右下に表示 |
